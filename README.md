
# CV Analyzer

AI-powered resume analyzer using Flask, React, and LLMs.

## Features
- Upload resume (PDF)
- Match with job description
- AI-generated insights (skills, gaps, suggestions)

## Tech Stack
- Backend: Flask
- Frontend: React
- AI: Hugging Face / Local LLM

## Setup

### Backend
cd backend  
python3 -m venv venv  
source venv/bin/activate  
pip install -r requirements.txt  
python app.py  

### Frontend
cd frontend  
npm install  
npm start  

## Note
Add your HF_TOKEN in environment variables.
